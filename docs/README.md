## ðŸ“– About This Project

The Electromagnetic Spectrum Explorer is an interactive educational tool for exploring electromagnetic radiation from radio waves to gamma rays. It features real-time visualization, unit conversion, and comprehensive educational content.

### Key Features
- Interactive spectrum visualization with logarithmic scaling
- Real-time unit conversion between wavelength, frequency, and energy
- Educational content for each electromagnetic region
- Responsive design for desktop and mobile
- Comprehensive theme system with light/dark modes
- Extensive testing suite ensuring accuracy

### Technology Stack
- **Frontend**: React 18, Vite, D3.js, Styled Components, Framer Motion
- **Testing**: Custom test runner with comprehensive physics validation
- **Development**: ESLint, modern JavaScript/JSX
